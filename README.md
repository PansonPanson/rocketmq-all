# 关于 rocketmq-all
本仓库旨在帮助开发者了解并深入学习 RocketMQ，未来的工作及更新计划包括以下几个方面：
- [ ] RocketMQ 基础学习（基础知识和代码示例）
- [ ] RokcetMQ 的服务部署和环境搭建 
- [ ] RocketMQ 的工业界实践合集
- [ ] RocketMQ 的原理剖析
- [ ] RocketMQ 的源码解析
- [ ] 由 RocketMQ 延伸出的知识
- [ ] 如何自己从零开始写一个精简的消息队列
- [ ] ……
